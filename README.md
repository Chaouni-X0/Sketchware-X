# Sketchware-X

سكتشوير إكس (Sketchware-X) هو نسخة مطورة ومحسنة من Sketchware Pro، تهدف إلى توفير تجربة تطوير تطبيقات أندرويد متكاملة مع دعم كامل للغة العربية ودمج تقنيات الذكاء الاصطناعي.

## المميزات الجديدة
- **واجهة ذكاء اصطناعي تفاعلية**: بناء تطبيقات كاملة من خلال الدردشة مع مساعد ذكي يشبه مانوس.
- **دعم API مخصص**: إمكانية إضافة مفاتيح API الخاصة بك لاستخدام نماذج الذكاء الاصطناعي المفضلة لديك.
- **تعريب كامل**: واجهة مستخدم عربية بالكامل لتسهيل عملية التطوير للمستخدمين العرب.
- **إصلاحات وتحسينات**: تم إصلاح العديد من الأخطاء البرمجية (مثل خطأ AboutActivity) وتحسين الأداء العام.
