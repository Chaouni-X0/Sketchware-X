# Sketchware-X (Edition X) 🚀

**Sketchware-X** هو الجيل القادم من بيئة تطوير التطبيقات للهواتف المحمولة، وهو نسخة مطورة كلياً من Sketchware Pro، تهدف إلى تمكين المطورين من بناء تطبيقات أندرويد احترافية باستخدام الذكاء الاصطناعي، البرمجة الهجينة، وأدوات البناء السحابية.

---

## 🌟 الميزات الثورية (Revolutionary Features)

### 🤖 محرك الذكاء الاصطناعي (AI Power)
- **AI App Generator**: إنشاء تطبيق كامل بضغطة زر من وصف نصي.
- **AI UI Generator**: توليد واجهات Material 3 حديثة ومعقدة.
- **Smart Auto-Fix**: اكتشاف وتصحيح الأخطاء البرمجية تلقائياً.
- **Error Explanation AI**: شرح الأخطاء البرمجية بأسلوب مبسط.
- **Auto Documentation**: توليد توثيق Javadoc/Kdoc تلقائياً.

### ✍️ المحرر الهجين (Hybrid Editor)
- **Blocks to Code**: التبديل اللحظي بين البرمجة بالبلوكات والأكواد المصدرية.
- **Multi-Language Support**: دعم Java, Kotlin, XML, C++, Python, JavaScript.
- **Sora Editor Integration**: محرر أكواد متقدم مع دعم التلوين البرمجي والتنسيق التلقائي.

### ☁️ البناء والسحابة (Cloud & Build)
- **Cloud Build**: بناء التطبيقات على السحابة لتوفير موارد الجهاز.
- **APK/AAB Export**: تصدير احترافي للتطبيقات للنشر على Google Play.
- **Performance Optimizer**: تحسين أداء التطبيق الناتج تلقائياً.

### 🛠️ أدوات المطور الاحترافية (DevTools)
- **Git Integration**: مدير GitHub مدمج لإدارة المشاريع والمزامنة.
- **API Manager**: إدارة وربط واجهات برمجة التطبيقات بسهولة وأمان.
- **Terminal Built-in**: طرفية أوامر مدمجة داخل التطبيق.
- **Live Preview**: معاينة الواجهات لحظياً أثناء التصميم.

### 🎨 التصميم والهوية (UI/UX)
- **Pure Black Mode**: وضع ليلي صرف (AMOLED) لتصميم عصري وأنيق.
- **Material Design 3**: دعم كامل لأحدث معايير التصميم من جوجل.
- **Theme Customizer**: تخصيص شكل التطبيق بالكامل.

# Sketchware-X (Edition X) 🚀 [English]

**Sketchware-X** is the next generation of mobile app development IDE, a completely revamped version of Sketchware Pro, designed to empower developers to build professional Android apps using AI, Hybrid Programming, and Cloud Build tools.

### Key Features:
- **AI Suite**: App/UI Generation, Smart Auto-Fix, and Auto Documentation.
- **Hybrid Editor**: Seamlessly switch between Blocks and Code (Java/Kotlin/C++/Python/JS).
- **Cloud Build**: Build and export APK/AAB on the cloud.
- **Git Integration**: Built-in GitHub manager for seamless collaboration.
- **Full Arabic Support**: 100% localized interface and blocks.

Developed by **Mohamed Chaouni**.
