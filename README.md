# Sketchware X 🚀

**سكتشوير إكس** هو النسخة الأكثر تطوراً واحترافية من Sketchware Pro، تم تطويره بواسطة **محمد الشاوني** ليكون بيئة تطوير متكاملة (IDE) على الهواتف الأندرويد، مدعوماً بتقنيات الذكاء الاصطناعي وأحدث أدوات البرمجة لعام 2026.

---

## 🌟 المميزات الرئيسية (Key Features)

### 🧠 ذكاء اصطناعي متكامل (AI Integration)
- **مساعد مانوس (Manus Assistant)**: واجهة دردشة متطورة لإنشاء المشاريع، إضافة البلوكات، وتعديل الواجهات بمجرد الوصف.
- **إصلاح الأخطاء التلقائي**: يقوم الذكاء الاصطناعي بتحليل أخطاء البناء واقتراح حلول برمجية فورية.
- **دعم Agent Code**: إمكانية إضافة مفاتيح API مخصصة وأكواد وكلاء لتخصيص سلوك الذكاء الاصطناعي.

### 🛠️ بيئة تطوير احترافية (Professional IDE)
- **دعم Kotlin و Jetpack Compose**: برمجة تطبيقات حديثة باستخدام أحدث لغات أندرويد.
- **محرر كود ذكي**: تلوين Syntax متقدم، إكمال تلقائي، ودعم كامل لـ Java/Kotlin/XML.
- **نظام بناء Gradle 8.13**: أداء بناء سريع ومستقر مع دعم SDK 36.

### 📦 إدارة متقدمة (Advanced Management)
- **مدير المكتبات (Dependency Manager)**: إضافة مكتبات مباشرة من Maven و GitHub.
- **تكامل GitHub**: رفع واستيراد المشاريع وإدارة الإصدارات بضغطة زر واحدة.
- **متجر الإضافات (Plugins)**: نظام يسمح بتثبيت إضافات ومكونات جديدة من المجتمع عبر GitHub.

### 🎨 تصميم عصري (Modern UI/UX)
- **دعم ConstraintLayout و RecyclerView**: بناء واجهات معقدة ومرنة بسهولة.
- **واجهة عربية بالكامل**: تجربة مستخدم عربية سلسة واحترافية.
- **أيقونة وهوية جديدة**: تصميم عصري يعكس قوة واحترافية التطبيق.

## 🚀 كيفية البناء من الهاتف (How to Build on Phone)

يمكنك بناء **Sketchware X** مباشرة من هاتفك باستخدام تطبيقات مثل **AndroidIDE** أو **AIDE**:

1. **استنساخ المستودع**:
   ```bash
   git clone https://github.com/Chaouni-X0/Sketchware-X.git
   ```
2. **الفتح في IDE**: قم بفتح المجلد المستنسخ في تطبيق البرمجة المفضل لديك على الهاتف.
3. **البناء (Build)**: انتظر تحميل المكتبات ثم اضغط على زر Build/Run.
